# 个人主页分类设计

## 目标

新增独立的“个人主页”作品分类，让学生上传包含完整静态网站的 ZIP，并自动生成可预览、分享和展示的网站链接。

## 上传限制

- 个人主页仅支持 ZIP 压缩包。
- ZIP 最大 100MB。
- 解压后全部文件最大 200MB。
- ZIP 内最多 300 个文件。
- 普通网页作品继续保持上传 20MB、解压后 100MB。

## 功能范围

- 复用现有 ZIP 安全检查、自动解压、HTML 入口识别、静态资源访问和删除清理逻辑。
- 复用现有在线预览、公开链接、二维码、下载和作品广场展示功能。
- 在学生控制台、教师控制台、学生明细、作品总览和作品广场中增加个人主页分类与统计。
- 不新增依赖，不修改数据库结构。

## 存储说明

原始 ZIP 与解压后网站文件都会保留，因此单个个人主页理论最大磁盘占用约为 300MB，不含可选封面图。
